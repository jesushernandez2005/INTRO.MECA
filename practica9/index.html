Brazo Robótico Bluetooth

Jesús León Hernández Martínez
Yael Alejandro Vázquez García
Ing. Mecatrónica
Octubre 25, 2024
Arduino Bluetooth Servomotores

- Resumen -

En esta práctica, se diseñó y programó una aplicación en MIT App Inventor para controlar un brazo robotizado de MDF compuesto por 5 servomotores, mediante conexión Bluetooth. La aplicación permite manipular el movimiento de cada servomotor individualmente, logrando un control preciso de las articulaciones del brazo en tiempo real. Los componentes principales incluyen el brazo MDF montado, los servomotores y un módulo Bluetooth HC-05 para la comunicación inalámbrica con el dispositivo móvil.

- Materiales -

- Brazo robótico de MDF

- Servomotores

- Módulo Bluetooth HC-05

- Microcontrolador Arduino

- Batería o fuente de alimentación

- Cables de conexión

- Protoboard

- Smartphone

- Introducción -

En esta práctica, se desarrollará un sistema de control para un brazo robótico de 5 grados de libertad, utilizando servomotores y comunicación Bluetooth. El objetivo es construir una interfaz de control móvil mediante una aplicación creada en MIT App Inventor, permitiendo el manejo remoto del brazo robótico a través de un dispositivo Android. Este tipo de sistema tiene aplicaciones en el ámbito de la robótica educativa y es un excelente ejemplo de cómo se puede integrar la programación, la electrónica y el diseño de interfaces móviles en un solo proyecto.

- Desarrollo -

Montaje del brazo robótico

Conectar los servomotores a los ejes del brazo de forma segura para evitar que se suelten durante el movimiento.

Conexión de los servomotores al Arduino

Alimentar los servomotores desde una fuente de energía externa para evitar sobrecargar el Arduino, ya que cada servomotor puede consumir una corriente significativa al moverse. Asegúrate de conectar las tierras de la fuente de alimentación y del Arduino para mantener una referencia común.

Programación del Arduino

Cargar un código en Arduino para recibir comandos a través del módulo Bluetooth. El código debe:
Configurar cada servomotor en el pin correspondiente.
Leer los comandos enviados por Bluetooth desde la aplicación móvil, que indicarán el ángulo deseado para cada servomotor.
Interpretar estos comandos y asignar el valor correspondiente a cada servomotor para lograr el movimiento en el ángulo deseado.

Conexión y configuración del módulo Bluetooth

Conectar el módulo Bluetooth (HC-05 o HC-06) al Arduino. El pin TX del módulo debe ir al pin RX del Arduino, y el pin RX del módulo al pin TX del Arduino.
Configurar el módulo Bluetooth para que pueda emparejarse con el dispositivo móvil.
Probar la comunicación con el módulo Bluetooth enviando comandos simples y verificando que el Arduino los reciba correctamente.

Desarrollo de la aplicación en MIT App Inventor

Crear una nueva aplicación en MIT App Inventor y diseñar una interfaz sencilla con botones o sliders para controlar el ángulo de cada articulación del brazo.

Configurar la funcionalidad de Bluetooth en MIT App Inventor para que la aplicación pueda emparejarse con el módulo Bluetooth.

Programar en MIT App Inventor para enviar un comando único cada vez que el usuario ajuste un slider o presione un botón. Este comando debe contener información de cuál servomotor ajustar y el ángulo deseado.

- Conclusiones -

El proyecto de control de un brazo robótico MDF con 5 servomotores mediante Bluetooth, usando una aplicación creada en MIT App Inventor, demuestra la efectividad de la integración entre hardware y software en sistemas de control remoto. La aplicación móvil facilita el envío de comandos al Arduino, permitiendo un control preciso y en tiempo real de cada articulación.
