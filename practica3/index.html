- Resumen -

En esta práctica, aprendimos a crear nuestra primera página web utilizando HTML. Comenzamos con una breve introducción al lenguaje de marcado HTML y su estructura básica, incluyendo las etiquetas principales como 'html', 'head', y 'body'. Posteriormente, diseñamos una página simple que incluía un título, párrafos de texto y enlaces.

Además, exploramos cómo estructurar el contenido de manera adecuada utilizando etiquetas semánticas como 'h1' para encabezados, 'p' para párrafos, y 'a' para hipervínculos. También aprendimos a insertar imágenes en la página con la etiqueta 'img'.

Al finalizar la práctica, entendimos la importancia de la organización del código y cómo un buen diseño de la estructura HTML es clave para crear páginas web funcionales y accesibles.

- Introducción -

El desarrollo web es una habilidad fundamental en el campo de la tecnología, y el lenguaje HTML (HyperText Markup Language) es el punto de partida para la creación de cualquier página web. HTML es un lenguaje de marcado utilizado para estructurar y organizar el contenido de las páginas web de manera que los navegadores puedan interpretarlo y mostrarlo correctamente.
En esta primera práctica, nos enfocaremos en los conceptos básicos de HTML y aprenderemos a construir una página web sencilla. Exploraremos la estructura fundamental de un documento HTML, utilizaremos las etiquetas más comunes y comprenderemos cómo éstas permiten organizar el contenido de una manera coherente y accesible para los usuarios. Esta introducción a HTML sentará las bases para el diseño web y nos preparará para avanzar en temas más complejos, como el uso de CSS.

- Conclusiones -

Al finalizar esta práctica, hemos logrado crear nuestra primera página web utilizando los conceptos básicos de HTML. Comprendimos la estructura fundamental de un documento HTML y aprendimos a utilizar etiquetas clave para organizar y presentar contenido en la web. Este ejercicio nos permitió ver cómo las instrucciones en HTML son interpretadas por los navegadores para mostrar textos, imágenes y enlaces.

La creación de una página web sencilla es solo el primer paso en el desarrollo web, pero nos proporciona una base sólida para profundizar en tecnologías más avanzadas como CSS, que nos permitirá estilizar nuestras páginas. Esta práctica también refuerza la importancia de la planificación y la organización del código para construir sitios web eficientes y bien estructurados.
