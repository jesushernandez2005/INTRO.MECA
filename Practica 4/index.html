Salidas digitales Arduino Uno

- Resumen -

Arduino se utiliza para desarrollar prototipos, controlar dispositivos como luces y motores, y crear sistemas interactivos. Es especialmente popular en el ámbito educativo, ya que permite a estudiantes y entusiastas aprender sobre programación y electrónica de manera práctica. Además, se puede emplear en proyectos de Internet de las Cosas (IoT), conectando dispositivos a la red para recopilar y compartir datos.

- Introducción -

En esta práctica veremos las funciones básicas de arduino, es una plataforma de hardware y software de código abierto que facilita la creación de proyectos electrónicos. Se compone de una placa de circuito impreso programable y un entorno de desarrollo (IDE) que permite escribir y cargar código, el objetivo de esta práctica es conocer las funciones basicas, por medio del código y salidas.

- Ejerciccio 1 y 4-

Este proyecto consiste en encender y apagar un LED a intervalos regulares, lo que permite entender conceptos básicos de programación y electrónica.

- Desarrollo -

Para comprender mejor, la placa fisica de Arduino UNO, son múltiples cicuitos integrados que permite que sea programable, empesaremos con un ejercicio básico,comúnmente conocido como "Blink", es una de las primeras experiencias que muchos entusiastas de la electrónica y programación realizan con Arduino.
Primero, es esencial preparar el entorno de trabajo. Para ello, se debe instalar el software de Arduino en una computadora. Este software proporciona un entorno de desarrollo donde se pueden escribir, compilar y cargar programas a la placa Arduino. Al abrir el IDE de Arduino, los usuarios se familiarizan con la interfaz, que incluye áreas para escribir código y herramientas para cargar programas a la placa.
A continuación, se procede a la conexión del hardware. En este paso, se necesita un LED, una resistencia y la placa Arduino. Se comienza conectando el ánodo del LED a uno de los pines digitales de la placa, como el pin 13, y el cátodo a una resistencia, que a su vez se conecta a tierra (GND). Esta configuración es crucial, ya que la resistencia limita la corriente que pasa a través del LED, protegiéndolo de daños. Es fundamental verificar que todas las conexiones sean correctas y estén firmes para evitar problemas durante la ejecución del programa. Una vez completada la conexión del hardware, se procede a configurar el programa.
Se abre un nuevo proyecto en el IDE de Arduino, donde se define el pin que se utilizará para el LED.
En la sección de configuración, se debe establecer este pin como salida. Este paso es esencial, ya que indica al Arduino que controlará un dispositivo externo, en este caso, el LED. La siguiente fase implica escribir la lógica que permitirá que el LED parpadee.
En el bucle principal del programa, se deben definir las instrucciones para encender y apagar el LED.
Aquí es donde se decide cuánto tiempo debe permanecer encendido y apagado, utilizando funciones que permiten hacer pausas en el código. Esta alternancia entre encender y apagar el LED es lo que genera el efecto de parpadeo, y es fundamental entender cómo funcionan los ciclos de tiempo en programación. Una vez escrito el código, el siguiente paso es cargarlo en la placa Arduino.
Para ello, se conecta la placa a la computadora mediante un cable USB y se selecciona el tipo de placa y el puerto correspondiente en el IDE. Luego, se utiliza la función de carga para transferir el programa a la placa. Es importante observar el proceso de carga, ya que cualquier error puede impedir que el LED parpadee correctamente. Finalmente, se verifica el funcionamiento del proyecto. Al observar el LED, debe comenzar a parpadear según el intervalo de tiempo que se estableció en el programa.
Si no ocurre así, es esencial revisar las conexiones y el código para identificar posibles errores. Una vez que el LED funcione correctamente, se puede experimentar con diferentes intervalos de tiempo, lo que permite a los usuarios entender mejor cómo ajustar el comportamiento de los componentes electrónicos.

- Ejerciccio 2 y 3-

En este ejercicio vamos a ver la capacidad que tiene de que sus salidas de estar encendidas o apagadas.

- Desarrollo -

En este ejercicio vamos a descubrir las salidas del Arduino UNO.
Las salidas URDUINO UNO pueden ser configurados a dos condiciones "HIGH", para cuando la salida expulsa un voltaje constante y "LOW", cuando la salida no expulsa voltaje, esto se puede configurar desde la plataforma de ARDUINO, primero tenemos que asignar una variable a un PIN, todo esto dentro de void setup, que es la configuración del programa, después en void loop, que es la parte de "void loop", la parte que se repite constante, les asignamos su salida, por la naturaleza de la practica no hay ninguna demostración visual más que el código.

- Resultados -

No se puede fisicamente pero al momento de programar uno de los pines del ARDUINO UNO en HIGH, la salida estará expulsando un voltaje estable constantemente y cuando este programada en LOW la salida no expulsará ningún voltaje.

- Conclusiones -

La placa ARDUINO UNO se puede utilizar para como una opción para programar y dar vida a nuestros circuitos por medio de salidas programables.
