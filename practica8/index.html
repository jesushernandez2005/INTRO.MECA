Control de servomotor mediante bluetooth

Jesús León Hernández Martínez
Yael Alejandro Vázquez García
Ing. Mecatrónica
Octubre 29, 2024
servomotor bluetooth

- Resumen -

El propósito de esta práctica es aprender a controlar un servomotor mediante una conexión Bluetooth utilizando un microcontrolador (como un Arduino) y una aplicación móvil personalizada creada en MIT App Inventor. Esta aplicación permite enviar comandos desde el teléfono al servomotor, configurando su posición de manera remota.

- Introducción -

En esta práctica, exploraremos el control de un servomotor mediante una aplicación móvil diseñada en MIT App Inventor. Este entorno de desarrollo simplifica la creación de aplicaciones móviles al permitirnos construir interfaces y programar funciones usando bloques gráficos. Con esta aplicación, podremos enviar comandos de control al servomotor a través de un microcontrolador, como el Arduino, que estará conectado al teléfono mediante un módulo Bluetooth. El servomotor es ideal para esta práctica ya que permite un control preciso de su posición en distintos ángulos, lo cual es útil en aplicaciones donde se requiere movimiento limitado y controlado, como en robótica, sistemas de apertura de puertas o incluso en proyectos de automatización casera. Mediante la comunicación Bluetooth entre el Arduino y la aplicación móvil, el usuario podrá definir la posición del servomotor desde su teléfono, facilitando el control remoto del dispositivo de manera intuitiva.

- Desarrollo -

Conexión del Servomotor: Conectamos el cable de señal del servomotor a un pin digital (por ejemplo, el pin 9) del Arduino, el cable de alimentación al pin de 5V y el cable de tierra a GND. Configuración del Módulo Bluetooth: El módulo Bluetooth se conecta al Arduino en los pines de comunicación serial (RX y TX), usando SoftwareSerial para no interferir con la comunicación USB del Arduino. Se configura a una velocidad de 9600 baudios, compatible con el módulo.

Diseño

Interfaz de la Aplicación: Creamos una interfaz con botones o un control deslizante (slider) que permita enviar posiciones específicas al servomotor. También añadimos el componente de conexión Bluetooth para enlazar el teléfono con el módulo Bluetooth del Arduino.

Bloques de Código: Programamos los bloques de código en MIT App Inventor para que, al presionar un botón o mover el control deslizante, se envíe el ángulo deseado en grados al módulo Bluetooth.

Nuestra aplicación se veria algo así.

- Resultados -

La aplicación móvil pudo enviar comandos específicos de posición al servomotor, y este respondió correctamente moviéndose a los ángulos solicitados (por ejemplo, 0°, 90°, 180°). Esto demuestra que la conexión Bluetooth permite una comunicación confiable y rápida con el microcontrolador.

La comunicación entre el módulo Bluetooth y la aplicación fue estable, sin desconexiones inesperadas en un rango de distancia de aproximadamente 1 a 3 metros. Esto es adecuado para aplicaciones de corto alcance en automatización doméstica o proyectos de robótica controlados a corta distancia.

- Conclusiones -

Esta práctica permite experimentar con el control remoto de dispositivos electrónicos utilizando Bluetooth y una aplicación móvil. La integración de Arduino y MIT App Inventor nos facilita diseñar aplicaciones personalizadas, controlando hardware externo de manera sencilla y flexible.
