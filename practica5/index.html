Arduino Entradas digitales

- Resumen -

En esta práctica, trabajamos con el microcontrolador Arduino para analizar el funcionamiento de las entradas digitales. Utilizamos un protoboard para armar los circuitos necesarios, conectando pulsadores a los pines de entrada del Arduino.

Programamos el microcontrolador para detectar cambios en las entradas digitales y registrar los estados "alto" y "bajo" de los pulsadores. El código se escribió utilizando el entorno de desarrollo Arduino IDE, donde configuramos los pines como entradas y utilizamos la función digitalWrite().

Durante el experimento, también exploramos cómo las señales digitales pueden utilizarse para activar otros componentes, como LEDs, en función de las entradas capturadas. Al finalizar la práctica, comprendimos cómo funcionan las entradas digitales y cómo programarlas en un entorno controlado, lo que nos prepara para trabajar con sensores más avanzados y sistemas de control.

- Materiales -

-1 Arduino uno

-1 Protoboard

-2 Pulsadores

-4 Resistencias de 220

-4 LEDs

-Cable Jumpers

- Introducción -

Las entradas digitales son fundamentales en la interacción de los microcontroladores con el mundo exterior. En este contexto, Arduino se ha convertido en una herramienta esencial para aprender y aplicar los conceptos de electrónica y programación. El uso de entradas digitales permite a un microcontrolador como Arduino detectar el estado de un interruptor, sensor o cualquier dispositivo que opere en dos estados: encendido/apagado o alto/bajo.
En esta práctica, nos enfocaremos en el análisis de entradas digitales utilizando Arduino y un protoboard. Armaremos circuitos simples para leer el estado de pulsadores y analizaremos cómo estas señales digitales pueden ser procesadas y utilizadas para controlar otros componentes. Esto nos permitirá entender cómo los sistemas digitales interactúan con su entorno, sentando las bases para aplicaciones más complejas en sistemas de control y automatización.

- Conclusiones -

A lo largo de esta práctica, logramos comprender el funcionamiento de las entradas digitales mediante el uso de un protoboard y un microcontrolador Arduino. Aprendimos a armar circuitos sencillos con pulsadores y a leer sus estados utilizando la función digitalRead(). También exploramos cómo las señales digitales pueden controlar otros dispositivos, como LEDs, y cómo el uso de resistencias pull-up y pull-down garantiza una lectura estable de las entradas.

Además, implementamos operaciones lógicas básicas como AND y OR en el código para evaluar combinaciones de entradas digitales. Estas operaciones nos permitieron ejecutar acciones específicas cuando se cumplen condiciones determinadas, como el encendido de un LED solo cuando dos pulsadores están activados (AND) o cuando al menos uno de ellos está presionado (OR). Esta integración de lógica digital refuerza el control sobre múltiples entradas, brindando mayor versatilidad en el diseño de sistemas de control.
